## 📋 Sobre o Projeto

Este é o **Projeto Fruteira** — um sistema simples de caixa desenvolvido em **Portugol** durante o curso de **Técnico em Desenvolvimento de Sistemas** no **SESI SENAI de Concórdia/SC**.

O programa simula o atendimento no caixa de uma fruteira: registra o produto comprado, calcula o valor total da compra e devolve o troco ao cliente de forma automática.

> ⚠️ **Aviso honesto:** este é um projeto simples e introdutório. Ele não tem a pretensão de ser sofisticado — tem a pretensão de ser **real**. Cada linha deste código representa um passo concreto na construção de uma nova trajetória profissional, partindo do zero na lógica de programação.

---

## 🎯 Objetivos de Aprendizagem

Com este projeto, foram praticados os seguintes conceitos fundamentais:

| Conceito | Aplicação no código |
|---|---|
| 📥 Entrada de dados | `leia()` para capturar produto, unidade e valor |
| 📤 Saída de dados | `escreva()` para exibir total e troco |
| 🔢 Tipos de variáveis | `real`, `inteiro`, `cadeia` |
| ➗ Operações aritméticas | Multiplicação e subtração para cálculo |
| 📚 Biblioteca padrão | `Matematica.arredondar()` para casas decimais |
| 🧩 Estrutura de funções | `funcao inicio()` como ponto de entrada |

## ⚙️ Como Baixar e Executar

### Pré-requisito: instalar o Portugol Studio

O Portugol Studio é o ambiente oficial para executar arquivos `.por`. É gratuito e leve.

1. Acesse **[portugolstudio.sourceforge.io](https://portugolstudio.sourceforge.io/)**
2. Clique em **Download** e baixe a versão para o seu sistema operacional (Windows, Linux ou macOS)
3. Instale normalmente seguindo o assistente

---

### Passo a passo para executar o projeto

**Opção A — Baixar apenas este arquivo**

1. Acesse o arquivo diretamente no GitHub:
   ```
   https://github.com/sangaletti35-ops/Estudos-Portugol/blob/main/Projeto-Fruteira/sistema_caixa.por
   ```
2. Clique no botão **Raw**
3. Use `Ctrl + S` para salvar o arquivo como `sistema_caixa.por`
4. Abra o **Portugol Studio**
5. Vá em **Arquivo → Abrir** e selecione o arquivo salvo
6. Clique no botão ▶ **Executar** (ou pressione `F6`)

---

**Opção B — Clonar o repositório completo**

> Requer o [Git](https://git-scm.com/) instalado na máquina.

Abra o terminal e execute:

```bash
# Clonar o repositório
git clone https://github.com/sangaletti35-ops/Estudos-Portugol.git

# Entrar na pasta do projeto
cd Estudos-Portugol/Projeto-Fruteira
```

Depois abra o arquivo `sistema_caixa.por` no Portugol Studio e execute com `F6`.

---

### O que esperar ao executar

```

Quantas unidades o cliente comprou? → Ex: 3

Qual o valor entregue pelo cliente? → Ex: 10.00
O troco do cliente deverá ser de: R$ 2.50
Volte sempre e deguste nossas delícias!
```
