# ⚽ Sistema de Placar de Jogo: Do Fluxograma ao Código

Este repositório contém um projeto desenvolvido em **Portugol**, com o objetivo principal de exercitar a lógica de programação através da leitura e interpretação de **fluxogramas**. 

O programa recebe o nome e a quantidade de gols de duas equipes, calcula a diferença no placar e determina o resultado da partida, informando se houve um empate, uma vitória normal ou uma goleada (diferença maior que 3 gols).

## 🗺️ O Fluxograma

A estrutura lógica do sistema foi construída seguindo rigorosamente o diagrama abaixo. Foi feito o mapeamento das decisões lógicas (losangos) para estruturas de controle e do processamento (retângulos) para os cálculos matemáticos.

## 🧠 Habilidades Praticadas

* **Interpretação de Diagramas Lógicos:** Tradução direta de um modelo visual para código algorítmico estruturado.
* **Estruturas Condicionais (Aninhamento):** Uso de `se`, `senao se` e `senao` para cobrir 100% dos cenários possíveis de forma eficiente.
* **Manipulação de Strings:** Inclusão de bibliotecas externas (biblioteca `Texto`) para tratamento de dados, utilizando a função `caixa_alta()` para padronizar e profissionalizar a exibição dos nomes das equipes no console.
* **Formatação de Saída (UX básica):** Ajustes de espaçamento em comandos de leitura (`leia`) e escrita (`escreva`) para melhorar a visualização do usuário final.

## 💻 Como executar o projeto

1. Faça o download e instale o [Portugol Studio](https://univali-lite.github.io/Portugol-Studio/).
2. Faça o clone deste repositório ou baixe o arquivo `.por`.
3. Abra o arquivo no Portugol Studio e clique em **Executar** (ou pressione Shift + F6).
4. Interaja com o programa através do console, inserindo os nomes dos times e os gols.
